SELECT Statement Basics

← Back to Dashboard

The SELECT statement is the most commonly used command in SQL. It retrieves data from one or more tables in a relational database. You can customize which columns to return, apply filters, sort results, and even join data from multiple sources.

📌 Basic SELECT Syntax

SELECT column1, column2
FROM table_name;

This fetches specific columns from a table.

🌟 Select All Columns

SELECT * FROM employees;

The asterisk (*) selects all columns.

🔍 Using Aliases

SELECT first_name AS "First Name", last_name AS "Last Name"
FROM employees;

Aliases make result columns more readable.

🎯 Distinct Values

SELECT DISTINCT department
FROM employees;

DISTINCT removes duplicate values in results.

🚀 Real-World Uses

  • Pull customer or employee lists from HR or CRM systems
  • Analyze product performance from sales tables
  • Build reports and dashboards with filtered views

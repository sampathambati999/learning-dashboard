GROUP BY & ORDER BY

← Back to Dashboard

GROUP BY and ORDER BY are clauses used to organize and sort the result set of an SQL query. GROUP BY is typically used with aggregate functions like COUNT or SUM, while ORDER BY controls the sorting of results.

  • ORDER BY can sort results by any column or alias in ASC (default) or DESC order.
  • You can use both together in the same query.

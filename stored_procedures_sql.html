Stored Procedures in SQL

← Back to Dashboard

A stored procedure is a named collection of SQL statements stored in the database. Procedures help encapsulate logic, reuse code, and improve performance by reducing network traffic.

üìÑ Creating a Stored Procedure

CREATE PROCEDURE GetEmployeeByDept (
  IN dept_name VARCHAR(50)
)
BEGIN
  SELECT * FROM employees
  WHERE department = dept_name;
END;

Defines a procedure with an input parameter to return employees by department.

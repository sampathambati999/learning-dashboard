ğŸ”§ What is Parameterization?

Parameterization in Informatica allows you to create flexible mappings and workflows that can dynamically adapt to different inputs, paths, values, or configurations without modifying the mapping logic itself.

ğŸ“ Types of Parameters and Variables

  • Mapping Parameters: Values assigned at the beginning and remain constant through the session.
  • Mapping Variables: Can change during session execution and hold a persistent value across runs.
  • Session Parameters: Parameters used in session-level configuration, e.g., source/target connection info.

ğŸ” Using Variables in Expression

You can reference and manipulate variables inside expression transformations using functions like SETVARIABLE, SETMAXVARIABLE, etc.

SETVARIABLE($$LastRunDate, CURRENT_DATE)

ğŸ§  Variable Evaluation Sequence

Informatica evaluates variables in the following order:

  • Pre-Session Variable Assignment
  • Parameter File
  • Initial Value
  • Value from Previous Session

WHERE, BETWEEN, IN, LIKE

← Back to Dashboard

The WHERE clause filters rows based on specific conditions. It's one of the most essential parts of writing precise queries in SQL. You can combine it with operators like BETWEEN, IN, LIKE, and logical connectors to match more complex patterns.

üîç Basic WHERE Clause

SELECT *
FROM employees
WHERE department = 'Sales';

Returns only employees from the Sales department.

üìÖ BETWEEN Operator

SELECT *
FROM orders
WHERE order_date BETWEEN '2023-01-01' AND '2023-01-31';

Filters records that fall between two date (or number) values.

üì¶ IN Operator

SELECT *
FROM products
WHERE category IN ('Books', 'Clothing', 'Gadgets');

Selects rows that match one of several specified values.

üî§ LIKE Operator

SELECT *
FROM customers
WHERE name LIKE 'J%';

Matches patterns (e.g., names starting with "J"). Use % as wildcard.

üß† Combining Conditions

SELECT *
FROM employees
WHERE department = 'IT' AND salary > 60000;

You can combine multiple conditions using AND, OR, and NOT.

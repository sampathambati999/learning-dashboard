📜 What is a Shell Script?

A shell script is a file containing a series of Unix commands. It automates repetitive tasks and can include variables, loops, conditionals, and functions.

#!/bin/bash

echo "Hello, World!"

🔤 Variables

Variables store data that can be reused in scripts.

name="John"
echo "Welcome, $name"

🧪 Conditionals

Use if statements to make decisions.

if [ $age -ge 18 ]; then
  echo "Adult"
else
  echo "Minor"
fi

🔁 Loops

Loops let you repeat actions.

for i in 1 2 3 4 5
do
  echo "Number $i"
done

🔧 Functions

Functions let you organize reusable blocks of code.

greet() {
  echo "Hi $1!"
}

greet "Alice"

🚀 Running a Script

Make the script executable and run it:

chmod +x script.sh
./script.sh

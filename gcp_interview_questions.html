Top 50 GCP Interview Questions

A curated list of commonly asked Google Cloud Platform interview questions and answers to help you prepare.

← Back to GCP Dashboard
1. What is GCP?
Google Cloud Platform is a suite of cloud computing services provided by Google for building, deploying, and managing applications.
2. List some core services in GCP.
Compute Engine, App Engine, Cloud Functions, GKE, BigQuery, Cloud Storage, Pub/Sub, etc.
3. What is a Project in GCP?
A Project is a container for resources, settings, permissions, and billing details.
4. Explain IAM in GCP.
Identity and Access Management controls access by assigning roles to users and service accounts.
5. What are service accounts?
Non-human accounts used to authenticate apps and services with permissions.
6. Difference between Compute Engine and App Engine?
Compute Engine offers VMs; App Engine is a serverless PaaS for applications.
7. What is Cloud Storage?
Object-based storage for unstructured data with multiple storage classes.
8. What is BigQuery?
A fully managed, serverless data warehouse for fast SQL analytics.
9. Explain VPC.
A Virtual Private Cloud is an isolated network environment with subnets and firewall rules.
10. What is Cloud Pub/Sub?
A messaging service for asynchronous communication between systems.
11. How does Cloud Build work?
Executes build steps defined in a config file, integrates with source code repositories.
12. What is Artifact Registry?
Stores container images and other build artifacts securely.
13. Difference between zonal and regional resources?
Zonal are confined to a single zone; regional are replicated across multiple zones.
14. What is Cloud Function?
A lightweight, event-driven serverless compute service.
15. Explain Dataflow.
A managed Apache Beam service for stream and batch data processing.
16. What is Cloud Run?
Serverless platform to run containers triggered by HTTP requests or events.
17. What is GKE?
Managed Kubernetes engine for container orchestration.
18. What is Preemptible VM?
Low-cost VM that can be shut down at any time.
19. Explain the Shared VPC concept.
Allows multiple projects to use a common network from a host project.
20. What is Cloud Spanner?
Globally distributed relational database with high availability and consistency.
21. How does Cloud Monitoring help?
Tracks metrics, uptime, and provides dashboards for system observability.
22. What is a Service Perimeter?
Used in VPC Service Controls to define a secure boundary around services.
23. What is a Stackdriver Logging?
Collects logs from GCP services for monitoring and auditing.
24. Difference between Cloud SQL and Firestore?
Cloud SQL is relational; Firestore is NoSQL document-based.
25. How does autoscaling work in Compute Engine?
Automatically adds/removes VMs based on load or custom metrics.
26. What is Identity-Aware Proxy (IAP)?
Controls access to web applications based on IAM policies and identity.
27. Can GCP connect to on-prem systems?
Yes, via VPN, Interconnect, or Transfer Appliance.
28. What is billing account in GCP?
Centralized account used to pay for usage across multiple projects.
29. What is CMEK?
Customer-Managed Encryption Keys used to control encryption of GCP data.
30. How are resources organized in GCP?
Organization > Folder > Project > Resource hierarchy.
31. What is Forseti?
An open-source tool for auditing GCP security policies and detecting misconfigurations.
32. Difference between regional and global load balancers?
Regional balances within a region; global supports multi-region routing.
33. What is coldline storage?
A low-cost storage class for infrequently accessed data.
34. What is gcloud CLI?
Command-line tool for managing GCP services and resources.
35. What is Cloud Shell?
Web-based shell environment preconfigured with CLI tools for GCP.
36. What is Cloud Armor?
A WAF and DDoS protection service to secure applications.
37. Explain use of Labels in GCP.
Key-value pairs for organizing and filtering GCP resources.
38. What is TPU?
Tensor Processing Unit, a Google-developed chip optimized for ML workloads.
39. How to secure API keys?
Use Secret Manager or environment variables with IAM policies.
40. Explain quotas and limits.
Controls the maximum use of GCP resources to prevent misuse.
41. What is Marketplace?
Catalog of preconfigured VM and container apps for quick deployment.
42. What are network tags?
Used to apply firewall rules and routes to specific VM instances.
43. What is Cloud Interconnect?
Provides high-speed physical connections between on-prem and GCP.
44. How to reduce BigQuery cost?
Use partitions, clustering, avoid SELECT *, limit scanned data.
45. What is Deployment Manager?
Infrastructure-as-code service to automate resource provisioning.
46. What is Cloud Endpoints?
API gateway that helps manage and secure APIs.
47. Difference between Cloud Tasks and Pub/Sub?
Tasks for async task queues; Pub/Sub for pub-sub messaging.
48. What is Cloud Identity?
Identity management for users and devices, integrates with IAM.
49. How to monitor billing costs?
Use Budgets, Billing Reports, and Cost Table in Cloud Console.
50. What is Anthos?
A platform for managing GKE clusters and workloads across hybrid and multi-cloud environments.

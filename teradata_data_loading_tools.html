Teradata Data Loading & Utility Tools

Overview and usage of various utilities in Teradata used for high-speed data loading, updating, and exporting.

üöÄ FastLoad

FastLoad is designed for quickly loading large volumes of data into an empty Teradata table using parallel sessions. It performs a two-phase load: data acquisition into work tables, followed by application into the base table. FastLoad does not allow duplicate rows, and cannot be used with tables having secondary indexes or triggers.

üì¶ MultiLoad

MultiLoad is used for inserting, updating, deleting, and upserting data into populated tables. It supports multiple DML operations and handles multiple input files. Unlike FastLoad, MultiLoad supports tables with secondary indexes, but it locks the target tables until completion, making it suitable for batch updates.

üîÅ TPump (Teradata Parallel Data Pump)

TPump allows continuous, low-volume row-by-row updates to Teradata tables. It supports INSERT, UPDATE, DELETE, and UPSERT operations using minimal locking, making it ideal for near-real-time data feeds. TPump works well with trickle-feed loading scenarios.

üîß Teradata Parallel Transporter (TPT)

TPT is a unified, high-performance data integration tool that combines the features of FastLoad, MultiLoad, TPump, and Export into a single scripting interface. It supports parallel job execution, metadata-driven operations, and integration with external data sources using connectors. TPT is recommended for modern, scalable loading and extraction jobs.

üì§ Teradata Export Utility

Export is used to unload large volumes of data from Teradata tables to flat files. It operates in parallel using multiple AMPs and supports delimited and fixed-width formatting. The Export utility is useful for data archival, migration, or reporting purposes.

‚úÖ Choosing the Right Utility

  • FastLoad: Best for initial bulk loads into empty tables
  • MultiLoad: Ideal for batch DML operations on existing data
  • TPump: Use for continuous, small-volume updates
  • TPT: Modern, unified utility for flexible and scalable operations
  • Export: Use to extract large datasets to flat files
